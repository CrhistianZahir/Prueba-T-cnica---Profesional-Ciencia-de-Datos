# 🚀 Prueba Técnica: Data Science & Engineering - Global Seguros (Ramo 69)

Este repositorio contiene la solución End-to-End para la prueba técnica del rol de Profesional en Ciencia de Datos. El proyecto abarca todo el ciclo de vida de los datos: desde el Análisis Exploratorio y Modelamiento Predictivo, hasta la propuesta de Arquitectura Tecnológica en la nube y visualización de KPIs.

## 📁 Estructura del Proyecto

* **`PruebaTecnica_DS_DiazCrhistian_20260529.ipynb`**: Notebook principal con el Análisis Exploratorio de Datos (EDA), limpieza, respuesta a preguntas de negocio y desarrollo de modelos de Machine Learning (Regresión y Clasificación) usando `scikit-learn`.
* **`Informe.pbix`**: Dashboard interactivo en Power BI con el panel ejecutivo de ventas y el monitor de riesgos.
* **`MODULO B Propuesta de Solucion.pdf`**: Diagrama de arquitectura de datos, gobierno del dato y roadmap de implementación ágil (6 meses) enfocado en optimizar el stack tecnológico actual.

## 🛠️ Stack Tecnológico Utilizado
* **Lenguajes:** Python, DAX, SQL (Diseño).
* **Librerías ML:** Scikit-Learn, Pandas, Seaborn, Matplotlib.
* **Visualización:** Power BI.
* **Arquitectura Propuesta:** SQL Server 2019, Power Apps, Power Automate, APIs.

## 🧠 Modelos de Machine Learning Implementados
1.  **Modelo A (Regresión):** Predicción de la Prima de Renovación. Se implementó un algoritmo *Gradient Boosting* logrando un alto R2 para la proyección de ingresos.
2.  **Modelo B (Clasificación - Churn):** Detección temprana de cancelación anticipada. Se ajustó el umbral de decisión para maximizar el *Recall* y priorizar la retención proactiva del negocio.
